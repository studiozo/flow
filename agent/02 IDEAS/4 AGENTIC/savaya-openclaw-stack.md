# IDEA: Savaya × OpenClaw Stack

Savaya is the next layer up from OpenClaw. OpenClaw = runtime. Savaya = memory + social layer.

Everything we built today (messages, contacts, files, memory, agent routing) IS the Savaya foundation. The fork of OpenClaw becomes the base infrastructure that Savaya builds on top of.

## The Vision
- OpenClaw handles execution (skills, channels, tools)
- Savaya handles memory (knowledge graph, relationships, temporal awareness, cross-agent memory)
- "Memory & social networking for AI" — agents sharing context about their humans (with consent)
